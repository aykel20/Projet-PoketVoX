# PoketVoX IA — PVX

Version complète améliorée de PoketVoX.

## Fonctionnalités

- Page de connexion / création de compte local
- Suivi par compte utilisateur
- Traduction IA de démonstration
- Phrases prédéfinies pour chaque situation
- Interface colorée et moderne
- Animations selon le thème choisi
- Historique par compte
- Favoris
- Statistiques et progression
- Support web Chrome inclus

## Lancer le projet

```bash
flutter pub get
flutter run -d chrome
```

## Note importante

La traduction IA est une simulation locale intelligente pour que le projet marche sans clé API. Le code est préparé pour remplacer facilement le service par une vraie API plus tard.
